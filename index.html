What is Expensei?

Expensei is a monthly subscription app that helps users track expenses, set short-term budgets, and reach savings goals. You can set custom targets (e.g., "spend under ¥10,000 this weekend"), record purchases, and get instant insights.

How It Works

  • Create an account
  • Set a budget or short-term spending goal
  • Track expenses manually or through receipts
  • View summaries and stay within limits
